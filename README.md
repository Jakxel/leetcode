# LeetCode Practice

Repository containing solutions to **LeetCode** problems, organized by
**programming language**, **topic**, and **difficulty**, with a focus on
clarity, efficiency, and clean coding practices.

## Repository Structure

The solutions follow this structure:
Language/
└── Topic/
└── Difficulty/
└── problem.cpp

Each file includes:
- Problem description
- Solution approach
- Time complexity
- Space complexity

## Covered Topics
- Arrays  
- Strings  
- Two Pointers  
- Sliding Window  
- Binary Search  
- Linked Lists  
- Trees  
- Graphs  
- Dynamic Programming  
- Math  

## Goal

This repository is intended to strengthen skills in:
- Problem solving
- Data structures
- Algorithms

The solutions are written as study material, technical reference, and
portfolio-ready code, following a consistent and well-documented format.
